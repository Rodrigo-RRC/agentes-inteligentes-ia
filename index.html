Módulo 1: Introdução aos Agentes Inteligentes

Objetivo: Entender o que é um agente inteligente de IA, suas aplicações e importância.

Um agente inteligente é um sistema que utiliza algoritmos de inteligência artificial para interpretar dados, interagir com usuários e executar tarefas de forma autônoma. Exemplos comuns incluem chatbots, assistentes virtuais e sistemas de recomendação.

Aplicações:

  • Saúde: Assistentes que auxiliam na triagem de pacientes e agendamento de consultas.
  • Alimentação: Sistemas para receber pedidos em pizzarias e lanchonetes.
  • Atendimento: Secretárias virtuais para diversas áreas (médicos, imobiliárias, etc.).

Exercício: Explique com suas palavras o que é um agente inteligente e cite uma aplicação prática que você considere relevante.

Módulo 2: Fundamentos de Inteligência Artificial e NLP

Conceitos Básicos: A inteligência artificial (IA) engloba métodos que permitem às máquinas aprender e tomar decisões. Dentro da IA, o Machine Learning e o Deep Learning são técnicas que possibilitam o reconhecimento de padrões e a tomada de decisões com base em dados.

Processamento de Linguagem Natural (NLP): É a área da IA que trata da interação entre computadores e linguagem humana. O NLP permite a análise, interpretação e geração de linguagem de forma natural. Exemplos de aplicações incluem tradutores automáticos, chatbots e sistemas de análise de sentimento.

Ferramentas e Modelos Gratuitos:

  • Hugging Face Transformers: Biblioteca para utilizar modelos de linguagem pré-treinados.
  • Mistral-7B-Instruct e LLaMA: Modelos open source para geração de texto.
  • Google Colab: Ambiente gratuito para execução de código Python.

Módulo 4: Desenvolvendo Seu Primeiro Chatbot com Hugging Face

Objetivo: Criar um agente inteligente simples usando a biblioteca transformers.

Explicação: Usaremos a função pipeline da biblioteca Hugging Face para gerar texto a partir de um modelo pré-treinado. O código a seguir ilustra como criar um chatbot que interage via linha de comando. 

from transformers import pipeline

# Inicializa o pipeline para geração de texto usando um modelo gratuito
chatbot = pipeline("text-generation", model="mistralai/Mistral-7B-Instruct")

while True:
    pergunta = input("Você: ")
    if pergunta.lower() == "sair":
        print("Até mais!")
        break
    resposta = chatbot(pergunta, max_length=200)[0]["generated_text"]
    print("Bot:", resposta)

Detalhamento: - pipeline("text-generation") cria um ambiente para gerar texto com base no modelo escolhido. - O loop while permite interagir continuamente com o usuário, e o comando if pergunta.lower() == "sair" interrompe a execução. - max_length define o tamanho máximo da resposta gerada.
